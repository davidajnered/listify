== Description ==



== Installation ==

Unzip the plugin, put it in the plugin folder and activate it. Go to the settings page and add a list. There are three ways to add a list to your site. Your can use the widget, you can use short tags and you can call listify from your theme.

= Widget =
Easy! Just drag the widget to the sidebar where you want to use it and select your list from the drop down. Done.

= Short Tag =
Also easy. The short tag looks like `[listify list="your-lists-name"]`
